구독 서비스 API 사용 가이드

1. 구독 등록

Endpoint:

POST /api/enroll

Request Body Example:

{
  "customerId": "123456",
  "planId": "plan_001"
}

Response:

{
  "status": "success",
  "message": "Subscription enrolled successfully."
}

2. 구독 활성화

Endpoint:

POST /api/activate

3. 모든 고객 정보 조회

Endpoint:

GET /api/customers

Response:

[
  {
    "id": 1,
    "customerId": "123456",
    "name": "John Doe",
    "email": "example@example.com"
  },
  ...
]

4. 특정 고객 정보 조회

Endpoint:

GET /api/customers/{customerId}

Response:

{
  "id": 1,
  "customerId": "123456",
  "name": "John Doe",
  "subscriptions": [
    {
      "id": "sub_001",
      "planName": "Premium Plan",
      "status": "active",
      ...
    },
    ...
  ]
}

5. 구독 Plan 정보 조회

Endpoint:

GET /api/plans

Response:

[
  {
    "id": "plan_001",
    "name": "Basic Plan",
    "price": 9.99,
    ...
  },
  ...
]
